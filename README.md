# Particle Portrait

10.000 partikulle olusan canvas tabanli fotograf animasyonu.

## Local

```bash
npm install
npm run dev
```

Site varsayilan olarak `http://127.0.0.1:5173/` adresinde calisir.

## Build

```bash
npm run build
```

Fotograf dosyasi `public/portrait.jpeg` icindedir. Baska bir fotografla denemek icin ayni dosyayi degistirmen yeterli.
